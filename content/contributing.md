---
title: Contributing
description: >
  How to contribute to the LUME ecosystem.
---

We are happy to accept contributions to the LUME ecosystem and use GitHub to manage software development efforts.
Drop in and submit a GitHub issue for problems or feature requests on the appropriate repo.
To contribute code, we accept PRs, but ask that an issue be opened for discussion of large contributions prior to submitting code.
Each of the packages within the LUME ecosystem has its own core developers (listed as "Maintain" or "Admin" on GitHub), and PRs are reviewed and merged at their discretion.
Small releases with bug fixes are made quickly, while larger architectural changes are reviewed and released quarterly.
