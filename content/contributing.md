---
title: Contributing
description: >
  How to contribute to the LUME ecosystem.
---

We are happy to accept contributions to the LUME ecosystem and use Github to manage software development efforts.
Drop in and submit a Github issues for problems or feature requests on the appropriate repo.
To contribute code, we accept PRs, but ask that an issue is opened for discussion of large contributions prior to submitting code.
Each of the packages within the LUME ecosystem has its own core developers (listed as "maintain" or "admin" on Github) and PRs are reviewed and merged at their discretion.
Small releases with bugfixes are made quickly with larger architectural changes reviewed and released quarterly.
